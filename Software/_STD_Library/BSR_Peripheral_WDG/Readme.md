RedBeanSprout WDG Example
========
* Author  : [Hom](http://about.me/Hom)
* Update  : 2015/02/19

Description
========
RedBeanSprout WDG 的範例程式，當按下 KEY_WU 延遲餵食 100ms，按下 KEY_BO 延遲餵食 1s，當延遲餵食超過 500ms 系統重新 Reset。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PA0  - KEY_WU  
PB2  - KEY_BO

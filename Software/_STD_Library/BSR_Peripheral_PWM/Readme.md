RedBeanSprout PWM Example
========
* Author  : [Hom](http://about.me/Hom)
* Update  : 2015/02/19

Description
========
RedBeanSprout PWM 的範例程式，按下按鍵 KEY 時，會改變 PWM 的 Duty Cycle，LED 同時會閃爍。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PA0  - KEY_WU  
PB2  - KEY_BO  

* PWM  
PA1  - TIM2 CH2 - PWM1  
PA2  - TIM2 CH3 - PWM2  
PA3  - TIM2 CH4 - PWM3  
PA6  - TIM3 CH1 - PWM4  
PA7  - TIM3 CH2 - PWM5  
PB0  - TIM3 CH3 - PWM6  
PB1  - TIM3 CH4 - PWM7  
PB6  - TIM4 CH1 - PWM8  
PB7  - TIM4 CH2 - PWM9  
PB8  - TIM4 CH3 - PWM10  
PB9  - TIM4 CH4 - PWM11
